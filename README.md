# darkwebinformer-bot
Bot de discord feito para mandar o RSS do darkwebinformer para um canal no discord

Este projeto apenas usa o discordgo como dependência e visa não usar bibliotecas da internet.

# Como rodar

Você precisa de um .env dentro da pasta src/ com as seguintes variaveis

```bash
TOKEN=<token do seu bot>
CHANNEL_ID=<ID DO CANAL PARA NOTICIAS>
```
